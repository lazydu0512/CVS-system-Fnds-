# 🚀 演唱会视频分享平台 - 快速启动指南

## 📋 系统要求

### 前端
- Node.js >= 16.0.0
- npm >= 8.0.0

### 后端
- JDK >= 11
- Maven >= 3.6.0
- MySQL >= 5.7

## 🎯 快速启动

### 1. 启动前端服务

```bash
# 进入前端目录
cd CVS-view

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev

# 前端将运行在 http://localhost:5173/
```

**前端已启动** ✅ - 当前正在运行中！

### 2. 启动后端服务

```bash
# 进入后端目录
cd CVS-port

# 使用Maven启动
mvn spring-boot:run

# 或者使用IDE（推荐）
# 在IDEA中打开CVS-port项目，运行主类
```

后端将运行在 `http://localhost:8080/`

### 3. 配置数据库

编辑 `CVS-port/src/main/resources/application.yml`:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/cvs_system?useUnicode=true&characterEncoding=utf8
    username: root
    password: your_password
```

## 📝 API接口

### 基础URL
```
http://localhost:8080/api
```

### 主要接口
```
GET  /videos              # 获取视频列表
GET  /videos/:id          # 获取视频详情
POST /videos              # 上传视频
PUT  /videos/:id          # 更新视频
DELETE /videos/:id        # 删除视频

POST /videos/:id/like     # 点赞视频
DELETE /videos/:id/like   # 取消点赞
POST /videos/:id/collect  # 收藏视频
DELETE /videos/:id/collect # 取消收藏

GET  /comments/:videoId   # 获取评论列表
POST /comments            # 发表评论
DELETE /comments/:id      # 删除评论

POST /auth/login          # 用户登录
POST /auth/register       # 用户注册
GET  /auth/user           # 获取用户信息
```

## 🐛 常见问题

### 1. 前端启动失败
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

### 2. 后端连接数据库失败
- 检查MySQL是否启动
- 检查数据库配置是否正确
- 检查数据库是否已创建

### 3. 跨域问题
后端已配置CORS，允许前端访问

### 4. 视频上传失败
- 检查文件大小限制
- 检查文件格式
- 检查上传目录权限
