## 🎨 设计规范

### 颜色规范
```css
--bili-pink: #fb7299      /* 主色 - 按钮、强调 */
--bili-blue: #00a1d6      /* 辅助色 - 链接 */
--bili-bg: #f4f4f4        /* 背景色 */
--bili-white: #ffffff     /* 卡片背景 */
--bili-text: #212121      /* 主文本 */
--bili-text-2: #505050    /* 次要文本 */
--bili-text-3: #999999    /* 辅助文本 */
--bili-border: #e7e7e7    /* 边框 */
--bili-hover: #f6f7f8     /* 悬停背景 */
```

### 组件规范
- **卡片**: 白色背景 + 圆角8px + 阴影
- **按钮**: 粉色主按钮 + 圆角6px
- **输入框**: 灰色背景 + 聚焦时白色 + 粉色边框
- **图标**: Element Plus Icons

### 布局规范
- **最大宽度**: 1400px
- **内边距**: 20px
- **卡片间距**: 20px
- **响应式断点**: 768px (移动端)
