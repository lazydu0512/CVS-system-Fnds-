# 🔧 图标问题修复说明

## 问题描述

在访问视频详情页时，出现以下错误：

```
SyntaxError: The requested module '/node_modules/.vite/deps/@element-plus_icons-vue.js?v=2b141e4a' 
does not provide an export named 'Heart' (at VideoDetail.vue:202:3)
```

## 问题原因

Element Plus Icons 库中**没有** `Heart` 和 `HeartFilled` 图标，但代码中尝试导入和使用这些图标。

## 解决方案

### 修改前
```javascript
import {
  Star,
  StarFilled,
  Heart,        // ❌ 不存在
  HeartFilled,  // ❌ 不存在
  Share,
  Loading
} from '@element-plus/icons-vue'

// 使用
<el-button :icon="isCollected ? HeartFilled : Heart">
  收藏
</el-button>
```

### 修改后
```javascript
import {
  Star,
  StarFilled,
  Collection,      // ✅ 使用 Collection 替代
  CollectionTag,   // ✅ 使用 CollectionTag 替代
  Share,
  Loading
} from '@element-plus/icons-vue'

// 使用
<el-button :icon="isCollected ? CollectionTag : Collection">
  收藏
</el-button>
```

## 注意事项

### 1. 图标导入方式

Element Plus 图标有两种使用方式：

**方式一：直接导入组件（推荐）**
```javascript
import { Star } from '@element-plus/icons-vue'

// 使用
<el-button :icon="Star">按钮</el-button>
```

**方式二：字符串方式（不推荐）**
```javascript
// 不需要导入

// 使用
<el-button icon="Star">按钮</el-button>
```

### 2. 动态图标

如果需要动态切换图标，使用方式一：

```javascript
<el-button :icon="isActive ? StarFilled : Star">
  {{ isActive ? '已点赞' : '点赞' }}
</el-button>
```

### 3. 图标大小和颜色

```vue
<template>
  <el-icon :size="20" color="#409eff">
    <Star />
  </el-icon>
</template>
```

## 其他可能的图标替代方案

如果需要心形图标，可以考虑：

1. **使用 Font Awesome**
```bash
npm install @fortawesome/fontawesome-free
```

2. **使用自定义 SVG 图标**
```vue
<template>
  <svg viewBox="0 0 1024 1024" width="1em" height="1em">
    <path d="M..." fill="currentColor" />
  </svg>
</template>
```

3. **使用 Element Plus 的其他图标**
   - `Collection` - 收藏夹
   - `CollectionTag` - 收藏标签
   - `Star` - 星星（推荐用于点赞）
