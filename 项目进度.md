## 🔧 技术栈

### 前端
- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI库**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia
- **HTTP客户端**: Axios

### 后端
- **框架**: Spring Boot
- **数据库**: MySQL
- **ORM**: MyBatis Plus
- **认证**: JWT

## 🚀 快速启动

### 前端
```bash
cd CVS-view
npm install
npm run dev
# 访问 http://localhost:5173/
```

### 后端
```bash
cd CVS-port
mvn spring-boot:run
# 访问 http://localhost:8080/
```
